# TTC
## ABOUT
本アプリケーション(以下TTC)は部活動支援ツールとして作成されたものです。技術スタックは以下の通りです。
- GAS(htmlのreturn, backend)
- Google SpreadSheet(as a database)

指定のGoogle SpreadSheetに部員情報および部活動日程情報を格納し、アクセスするたびにGASを用いてデータをreturnします。

> GASで処理を行う都合上、通常のWEBポータルに比べて動作が遅い場合があります。あまりにもデータ取得に時間がかかる場合はデータ取得アルゴリズムをアップデートします。

## HOW TO USE
TTCは`SCHEDULE` `MEMBER` `INFO`の3つのページに分かれています。それぞれの利用方法は以下のドキュメントを読んでください。

### `SCHEDULE`
下部ナビゲーションバー左側のカレンダーアイコンをクリックまたはタップすることで移動できます。このページでは以下の機能を利用することができます。

##### 1.日程追加
ページ右上にある青いカレンダープラスアイコンをクリックまたはタップすると、日程追加用フォームが表示されます。各入力データの規則は以下の通りです。

|名称|データタイプ|
|---|---|
|タイトル|文字列|
|日付|文字列|
|開始時間|時間|
|終了時刻|時間|
|備考|文字列|

これらを入力したのち、フォーム下部「スケジュールを追加」ボタンを押すことで送信できます。

##### 2-1.登録されている予定の参照
TTCにアクセスすることで自動で表示されるデータです。以下の項目2-2, 2-3と関連して、登録済みの予定に関する情報を見ることができます。

##### 2-2.詳細確認・編集
各予定の「詳細」ボタンを押すことで、その予定についての詳細情報を確認することができます。**欠席者の一覧を参照することが可能**です。

さらに、各項目のデータを書き換えてフォーム下部「スケジュールを変更」ボタンを押すことで、その**予定の情報を更新**することができます。

##### 2-3.欠席連絡
各予定の「欠席連絡」ボタンを押すことで、その予定に対して欠席・遅刻・早退の申請を行うことができます。各入力データの規則は以下の通りです。

|名称|データタイプ|
|---|---|
|申請理由|文字列|

フォーム中段「申請送信」ボタンを押すことで送信できます。

また、フォーム下部「この日程に対する申請を削除します」ボタンを押すことで、**その予定に対する欠席・遅刻・早退の申請を取り消す**ことができます。**自分以外のユーザーの申請に対しては動作しません。**

### `MEMBER`
下部ナビゲーションバー央側のユーザーアイコンをクリックまたはタップすることで移動できます。このページでは以下の機能を利用することができます。

##### 1.部員の追加
ページ右上にある青いユーザーラスアイコンをクリックまたはタップすると、部員追加用フォームが表示されます。各入力データの規則は以下の通りです。

|名称|データタイプ|
|---|---|
|学籍番号|整数|
|固有学籍番号|整数|
|氏名|文字列|

これらを入力したのち、フォーム下部「送信」ボタンを押すことで送信できます。

##### 2-1.登録されている部員の参照
TTCにアクセスすることで自動で表示されるデータです。学籍番号、氏名、メールアドレス、権限の情報を見ることができます。

##### 2-2.部員情報更新
<ins>2-1.登録されている部員の参照</ins>で確認したデータの最右列のユーザーペンアイコンをクリックまたはタップすることで、部員情報更新用フォームが表示されます。

各項目のデータを書き換えてフォーム下部「更新」ボタンを押すことで、その**部員の情報を更新**することができます。

### `INFO`
下部ナビゲーションバー右側のインフォメーションアイコンをクリックまたはタップすることで移動できます。このページでは以下の機能を利用することができます。

##### 1.プライバシーポリシーに関する表示
プライバシーに関する規定を表示します。
##### 2.ログイン情報の表示
> 今後アップデートにより実装予定です。
##### 3.管理者ページ
> 今後アップデートにより実装予定です。
